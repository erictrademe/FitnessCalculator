# Fit(X)

Fit(X) is an android app offering a range of health and fitness calculators.


#### Current calculators

##### Body
- Calorie intake
- Macro breakdown
- Water intake
- Ideal weight
- Body mass index
- Bodyfat
- Ideal physique

##### Strength
- Wilks
- One rep max
- Strength standards


#### Feature backlist
